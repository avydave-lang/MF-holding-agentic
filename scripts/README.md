# scripts/

This directory contains per-site Playwright scrape scripts generated by the Discovery Agent.

## Naming convention

Each file is named `<domain_hash>.py` where `domain_hash` is the SHA-256 (first 32 chars)
of the competitor domain, as computed by `app.memory.domain_hash(url)`.

## Required interface

Every script **must** expose a `run() -> dict` function at the module level.

The returned dict must include at minimum:
- `product_name` (str)
- `price` (str | float | int)
- `url` (str)

Optional metadata fields understood by the pipeline:
- `_confidence_raw` (float 0–1): override the default confidence calculation
- `_selectors_used` (list[str]): CSS selectors used, for fingerprinting
- `_raw_html` (str): full page HTML, used to compute dom_hash

## Example

```python
from playwright.sync_api import sync_playwright


def run() -> dict:
    with sync_playwright() as p:
        browser = p.chromium.launch(headless=True)
        page = browser.new_page()
        page.goto("https://example.com/product/widget")
        name = page.locator("h1.product-title").inner_text()
        price = page.locator("span.price").inner_text()
        browser.close()

    return {
        "product_name": name,
        "price": price,
        "url": "https://example.com/product/widget",
        "_selectors_used": ["h1.product-title", "span.price"],
    }
```
